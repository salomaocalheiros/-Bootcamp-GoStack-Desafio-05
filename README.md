## Tecnologias

- JavaScript

## Como rodar

### Pré-requisitos

- Node.js 18+ e npm

### Instalação

```bash
git clone https://github.com/salomaocalheiros/-Bootcamp-GoStack-Desafio-05.git
cd -Bootcamp-GoStack-Desafio-05
npm install
```

### Execução

```bash
npm start
```
